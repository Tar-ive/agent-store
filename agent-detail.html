Installation & Setup

Requirements

  • Python 3.8 or higher
  • LangChain library (latest version)
  • OpenAI API key or compatible LLM provider
  • Internet connection for external API calls

Install via pip

pip install langchain-react-research-agent

Environment Setup

# Set your API key
export OPENAI_API_KEY="your-api-key-here"

# Or create a .env file
echo "OPENAI_API_KEY=your-api-key-here" > .env

API Documentation

Initialization

from react_research_agent import ReActResearchAgent

# Initialize the agent
agent = ReActResearchAgent(
    model="gpt-4",
    temperature=0.7,
    max_iterations=10,
    verbose=True
)

Basic Usage

# Simple research query
result = agent.research(
    query="What are the latest developments in quantum computing?",
    max_sources=5,
    include_analysis=True
)

print(result.summary)
print(result.sources)
print(result.analysis)

Configuration Options

Parameter Type Default Description
model string "gpt-4" LLM model to use
temperature float 0.7 Model temperature (0-1)
max_iterations integer 10 Maximum reasoning iterations

Usage Examples

Example 1: Market Research

# Market research for electric vehicles
research_task = """
Analyze the electric vehicle market in 2024, including:
1. Market size and growth projections
2. Key players and their market share
3. Consumer adoption trends
4. Regulatory landscape
5. Future outlook
"""

result = agent.research(
    query=research_task,
    max_sources=8,
    include_analysis=True,
    save_to_file="ev_market_analysis_2024.md"
)

print(f"Research completed in {result.iterations} iterations")
print(f"Sources used: {len(result.sources)}")
print(f"Analysis length: {len(result.analysis)} characters")

Example 2: Academic Literature Review

# Literature review on machine learning in healthcare
literature_review = """
Conduct a comprehensive literature review on the application of 
machine learning in medical diagnosis. Focus on:
- Recent breakthroughs (2020-2024)
- Accuracy improvements over traditional methods
- Real-world implementations
- Challenges and limitations
- Future research directions
"""

result = agent.research(
    query=literature_review,
    max_sources=12,
    include_analysis=True,
    academic_focus=True,
    save_to_file="ml_healthcare_literature_review.md"
)

# Generate summary for quick review
summary = agent.summarize(result.analysis, max_length=500)
print("Literature Review Summary:")
print(summary)

Example 3: Competitive Analysis

# Competitive analysis for AI writing tools
competitors = ["OpenAI GPT", "Claude", "Gemini", "Jasper", "Copy.ai"]

analysis_tasks = []
for competitor in competitors:
    task = f"""
    Research {competitor} as an AI writing tool competitor:
    - Pricing and plans
    - Key features and capabilities
    - Target market and use cases
    - Strengths and weaknesses
    - User reviews and feedback
    """
    analysis_tasks.append(task)

# Run parallel research
results = agent.research_parallel(
    queries=analysis_tasks,
    max_workers=3,
    save_individual_files=True
)

# Generate comparative analysis
comparison = agent.compare_results(results)
print(comparison)
